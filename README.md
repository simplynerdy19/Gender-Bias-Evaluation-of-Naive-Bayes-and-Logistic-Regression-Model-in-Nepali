# Gender-Bias-Evaluation-of-Naive-Bayes-and-Logistic-Regression-Model-in-Nepali
Evaluating gender bias in Nepali COVID-19 sentiment classification using Naive Bayes and Logistic Regression. Implements Identity Term Sampling (ITS) and GS-CDA adapted from Sobhani &amp; Delany (2024) for a low-resource language context.
