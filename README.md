## Gender-Bias-Evaluation-of-Naive-Bayes-and-Logistic-Regression-Model-in-Nepali

### Overview

Gender bias in natural language processing (NLP) systems has been extensively studied for high-resource languages, yet remains largely unexplored for low-resource languages such as Nepali. This study evaluates gender bias in binary sentiment classification for Nepali by comparing two classical machine learning models i.e., Multinomial Naive Bayes and Logistic Regression, trained on a COVID-19 labelled tweet dataset. Feature extraction was performed using TF-IDF vectorization across unigram and unigram+bigram configurations. To identify gender in the dataset without relying on synthetically generated test sets, this study adapts Identity Term Sampling (ITS) and Gender-Selective Counterfactual Data Augmentation (GS-CDA) from Sobhani and Delany (2024) for Nepali. Gender bias is quantified using True Positive Rate (TPR) and Class Gap metrics. Results show that bigram features improve overall classification performance for both models. In bias evaluation, all four model configurations exhibit a consistent male advantage in positive sentiment classification, while patterns for negative sentiment are less consistent across architectures. The Logistic Regression model demonstrates more stable bias measurements owing to its conservative classification strategy. To the best of the author's knowledge, this is the first study to evaluate gender bias in Nepali sentiment classification using real dataset instances, establishing a reproducible baseline for future fairness evaluation research in low-resource South Asian languages.
